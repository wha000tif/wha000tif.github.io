Python正则表达式学习

正则表达式前前后后看过几次，但是因为之前没怎么用到，所以经常性遗忘。这里做个笔记记录下。
首先，在Python中正则表达式大概分为一下几个部分:

  1. 元字符
  2. 模式
  3. 函数
  4. re内置对象使用方法
  5. 分组用法
  6. 环视用法

一、 元字符

匹配字符

.               匹配任意的字符，但是不包括换行符
\d              匹配数字
\s              匹配任意的空白字符相当于 [\t\n\r\f\v]
\w              匹配字母数字下划线或者汉字
\W              匹配任意非(字母，数字，下划线，汉字)的字符
\S              匹配非任意空白字符相当于[^\t\n\r\f\v]
\D              匹配任意非数字字符相当于[^0-9]

表示重复

*               重复0次或者更多次
+               重复一次或者多次
?               重复0次或者一次
{n}             重复n次
{n,}            重复n次或者更多次
{n,m}           重复n-m次

匹配位置

\\b             匹配单词的开始或者结束
^               匹配字符串的开始
$               匹配字符串的结束
\G              上一个匹配的结尾(本次匹配的开始)
\A              字符串开头(类似^,但是不受处理多行选项的限制)
\Z              字符串结尾或者行尾(不受处理多行选项的影响)
\z              字符串结尾类似$，不受多行处理选项影响

字符类

字符类
\0XX          8进制值XX所表示的字符
\xXX          16进制值XX所表示的字符
\uXXXX        16进制值XXXX所表示的unicode字符
\t            Tab
\n            换行符
\r            回车符
\f            换页符
\e            Escape
\cN           ASCII码控制字符
\p{name}      unicode中命名为name的字符类，例如\p{IsGreek}  

陈列  
[abcde]       匹配一个盘[]中的一个字符
[.?!]         匹配给定的一个符号  

范围  
[]            字符集，一个字符的集合，可以匹配其中任意一个
|             逻辑表达式或， a|b 可以匹配a也可以匹配b  
[0-9]         匹配0-9的数字，等同于\d
[a-z]         匹配a-z的所有小写字母
[a-zA-Z]      匹配所有字母等同于
[0-9a-zA-Z_]  等同于\W
[^aeiou]      匹配除了aeiou几个字符外的其他所有字符

分组

(...)         分组，默认为捕获，即被分组的内容可以被单独取出，默认的每个分组都有一个索引，从1开始
(?:...)       分组不捕获模式，计算索引时会跳过这个分组
(?P<name>...) 分组的命名模式，取出分组中的内容时可以使用索引也可以使用name
(?P=name)     分组的引用模式，可以在同一个正则表达式中引用前面命名过得表达式
(?#...)       注释，不会影响正则表达式的其他部分
(?=...)       顺序肯定环视，表示所在位置右侧能匹配括号内正则(但是不包括右侧匹配到的内容)
(?!...)       顺序否定环视，表示所在位置右侧内容应是不能匹配括号内正则
(?<=..)       逆序肯定环视，表示所在位置的左侧可以匹配括号内正则(但是布包括座侧匹配到的内容)
(?<!..)       逆序否定环视，表示所在位置的作侧内容应是不能匹配括号内正则
\number       匹配和前面索引为number的分组捕获到的内容一样的字符串
(?(id/name)yes|no)    若前面id/name指定的分组匹配成功，则执行yes处的匹配，否则执行no处的匹配

函数

re.findall(pattern, string)       找到之后返回符合组的列表，组开头有?:则返回整个
re.search(pattern, string)        search 扫描整个字符串并返回第一个成功的匹配match对象。
re.match(pattern, string)         re.match 尝试从字符串的起始位置匹配一个模式，如果不是起始位置匹配成功的话，match()就返回none。
re.finditer(pattern, string)      找到re匹配的所有的子串，并把它们作为一个迭代器返回
re.split(pattern, string)         分割字符串
re.sub(pattern, repl, string, count=0)     替换count个，0为所有
re.subn(pattern, repl, string)            功能同上，返回一个元组(替换完的字符串，替换次数)

match对象的方法

group()             返回匹配到的字符串
start()             返回匹配到的起始位置
end()               返回匹配到的结束位置
span()              返回一个元组(start，end)
group(num/name)     返回指定组匹配到的内容
